# Эксперимент с движущимся кружком

Эксперимент для изучения реакции на движущиеся объекты.

## Структура проекта

- `index.html` - главная страница с выбором эксперимента
- `index-file1-speed1x.html` - эксперимент 1: 100 мишеней со скоростью 1x (60 px/сек)
- `index-file2-speed2.5x.html` - эксперимент 2: 100 мишеней со скоростью 2.5x (150 px/сек)

## Деплой на Vercel

1. Установите Vercel CLI (если еще не установлен):
```bash
npm i -g vercel
```

2. Войдите в Vercel:
```bash
vercel login
```

3. Деплой проекта:
```bash
vercel
```

4. Для продакшн деплоя:
```bash
vercel --prod
```

## Доступ к экспериментам

После деплоя файлы будут доступны по следующим URL:
- Главная страница: `https://your-project.vercel.app/`
- Эксперимент 1: `https://your-project.vercel.app/index-file1-speed1x.html`
- Эксперимент 2: `https://your-project.vercel.app/index-file2-speed2.5x.html`

## Формат данных

Результаты скачиваются в формате CSV с разделителем точка с запятой (для русского Excel).

Колонки:
- Триал
- Скорость(px/кадр)
- Скорость(px/сек)
- Отклонение
- Направление (раньше/позже/идеально)
- ВремяРеакции
- Позиция
- ТипОтвета
- ИдеальноеПопадание
- Время
